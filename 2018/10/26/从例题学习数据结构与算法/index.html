从例题学习数据结构与算法

发表于 2018-10-26 | 分类于 数据结构与算法

前言

以前总是断断续续学习数据结构和算法方面的知识，学了忘–忘了学，无限死循环。这次打算从编程题的角度来学习这方面的知识，这样也许可能会避免学了后遇到编程题还是无从下手的情况。最近在线笔试了几家公司，对算法的要求还是挺高的。这些题目虽然不是特别难，但是想要在短时间把问题AC，还是有一定难度的。

这篇博客从一开始就是打算按照一本数据结构和算法书来写的，在《数据结构与算法》分类下，接下来会有很多篇博客记录自己学习各种数据结构和算法的历程。在大学跨专业考研时，自学了严蔚敏的《数据结构》(当时的408参考书目)、读研后又选修了学校开设的 “算法设计与分析” 这门课程、非常稀少的时间会翻开那本厚厚的《算法导论》(太厚了，买了也不怎么会看，可以当做信仰书籍吧 哈哈)、偶尔心血来潮打开各大OJ网站花几个小时解一道题。以上就是我这方面的基础，也是这些博客的前提。

再次说明一下，这些博客是基于以上基础来记录新的学习历程，进行系统的学习数据结构与算法。因此，博客的顺序不会是像其他书籍那样，先写线性结构的栈、队列等等。以下是这个系列博客在解题过程中提供的语言版本：

  • Python
  • C++
  • Java

以及题目的主要来源(可能不局限于此)：

  • LeetCode
  • hihocoder
  • 在线学习课程的作业题目
  • 各大名企校招、社招题目
