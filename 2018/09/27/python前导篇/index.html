1.Python的几种实现方式[1]

  • Cpython

    python的官方版本，也就是从官网发布的最广泛的版本。使用C语言实现，新的语言特性一般也最先出现在这里。CPython实现会将源文件（py文件）转换成字节码文件（pyc文件），然后运行在Python虚拟机上。

  • Jpython

【引】什么是Cython？

由于python代码在某些情况下运行速度低，而且主流的python实现方式是使用C语言实现的。因此出现了一些看不惯python那么慢的人，将python源代码编译成C语言代码。但是又不仅仅如此！他们 (https://cython.org/) 的编译器不仅仅可以编译python编程语言，还支持一种C+Python风格的Cython编程语言。

因此Cython是python的超集。

Cython在线文档、性感教学：http://docs.cython.org/en/latest/

  • 这个东西是奇葩！

    个人倾向于将Cython定义为python和C的结合体。虽然是编译器、一种语言(结合体语言)，但是又没有编译器，编译方法是使用基于python的第三方库： pip install Cython

    • 新建一个helloworld.pyx 文件：

    • 在helloworld.pyx 文件的同级目录下新建setup.py

    • 使用命令行运行setup.py文件：

      python setup.py build_ext --inplace

      编译可以得到.C和.pyd文件等。后面的具体应用参见其在线文档http://docs.cython.org/en/latest/

Python相关的文件格式(编译与反编译)

        ①.py 是源文件；     ②.pyc 是源文件编译后的文件

        ③.pyo是源文件优化编译后的文件

        ④.pyd是其他语言写的python库(python的动态链接库, 本质上和dll没区别,只是后缀名不同)

​ .pyd文件并不是使用python编写而成，.pyd文件一般是其他语言编写的python扩展模块。（之前又在网上看到过有关解释，.pyd文件是用D语言按照一定的格式编写，并处理成二进制文件。那么什么是D语言呢？？它是c/c++的综合进化版，不仅具有二者的全部优点，而且整体性能更佳，但是其抽象程度高。）

如果不希望源代码被看到，则可以将py源代码编译成pyc或者pyo格式的编译后的字节码文件。

注意：在python3.5版本之后，不再使用.pyo文件后缀，而是生成文件名类似“xxx.opt-n.pyc的文件。

可以在cmd/dos窗口使用python xx.pyc来执行编译后的代码

  • 编译的方法是使用python提供的内置模块：py_compile

  • 如果想要编译指定目录下的所有python源代码文件，则使用内置模块：compileall

  • 使用命令进行编译

    python -m py_compile fileName.py # 把单个.py文件编译为字节码文件

    python -m compileall fileName.py # 也可以使用compileall编译单个文件
    python -m compileall SourcePath # 批量生成字节码文件，SourcePath 是包含.py文件名的路径

  • 使用命令进行优化编译为.pyo文件

    python -O -m py_compile fileName.py # 把单个.py文件编译为字节码文件

    python -O -m compileall fileName.py # 也可以使用compileall编译单个文件
    python -O -m compileall SourcePath # 批量生成字节码文件，SourcePath 是包含.py文件名的路径

    注意： -O(大写字母O)也可以使用-OO

Tips: 可以使用uncompyle6对pyc文件进行反编译。：

安装：pip install uncompyle6 位置：…./Scripts/uncompyle6.exe

使用：uncompyle6 xx.pyc > fname.py

python执行系统命令的四种方法

  • os.system(command=””)

    类似 C 中的 system() 函数，command 为os命令；
    返回值在不同os上不同，Linux 返回值是执行命令的 exit 值，Window 返回值为执行命令后 shell 的返回值；
    该方法无法获取到运行命令的输出；如果py文件在窗口执行 会自动打印结果。

  • os.popen(cmd, mode=”r”, buffering=-1)

    参数详解：

    cmd：调用命令

    mode：返回 file 对象的模式，默认为 “r” 只读模式；”w”时 可以将返回值写到文件流中。

    buffering：缓冲区大小，默认 -1 为无限制

类方法前面加单下划线和双下划线：

​ “_”：以单下划线开头的表示的是protected类型的变量，即只能允许其本身与子类进行访问；同时表示弱内部变量标示，如，当使用”from moduleName import *”时，不会将以一个下划线开头的对象引入。

​ “__”：双下划线的表示的是私有类型的变量。只能是允许这个类本身进行访问了，连子类也不可以，这类属性在运行时属性名会加上单下划线和类名。

​ 私有方法: 实例._类名__方法名()

python函数定义时，函数的注释

  1. 静态注释

    • 参数注释的一般规则是参数名后跟一个冒号:，然后再跟一个expression。冒号后面表示需要传入参数的数据类型。

    • -> 后面是函数返回值类型的注释(expression)

    • 注释的这个expression可以是任何形式。

      int，annotation可被保存为函数的attributes。

    在python3.6及更高的版本。支持为变量添加注解，来显示这个变量的类型。

    同样也不是强制的规定，而是提醒。例如：def f(a：int = 100)

  2. 动态注释

    动态注释的原理，就是在函数中或者装饰器中动态的增加 删除 更改 注释内容。

python中字符(串)与数值

ord(c):参数是长度为1的字符串，简称字符。当参数为统一对象时（unicode object），返回能代表该字符的统一编码，当参数为8比特的字符串时，返回该字节的值。例如，ord(‘a’)返回整形数值97，ord(u’\u2020’)返回8224。

chr(i)：返回一个字符，字符的ascii码等于参数中的整形数值。例如chr(97)返回字符’a’，该方法是ord()的反方法。参数必须是0-255的整形数值，否则会抛出valueError错误。
